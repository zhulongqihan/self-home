# Mini Order ☕️

> 一个轻量级的微信小程序点单系统模板，灵感来源于主流连锁饮品店的下单体验。

适用于学习微信云开发、构建个人/小型店铺点单工具，或作为二次开发的基础脚手架。

---

## ✨ 功能特性

- 📱 **顾客端**：商品浏览、规格选择、购物车、订单跟踪、个人中心
- 🧑‍🍳 **管理端**：商品管理（CRUD）、分类管理、订单接单与状态流转、数据看板
- 🎨 **双主题切换**：暖阳大地（默认）/ 简约云白，支持日夜自动切换
- 📦 **订单全流程**：待接单 → 已接单 → 制作中 → 配送中 → 已送达 → 待评价 → 已完成
- 🔔 **消息推送**：基于微信订阅消息，订单状态变更实时触达
- 🛠️ **后台高度可配置**：店铺名、分类、价格单位、文案、主题色等均可在管理端修改，无需改代码
- ☁️ **零运维**：基于微信云开发，免服务器、免备案、免域名

## 🚀 快速开始

### 环境要求
- 微信开发者工具（最新稳定版）
- Node.js 16+
- Git

### 安装步骤

1. **克隆仓库**
   ```bash
   git clone <repo-url>
   cd <repo-folder>
   ```

2. **配置 AppID**
   - 在 [微信公众平台](https://mp.weixin.qq.com/) 申请小程序 AppID
   - 复制 `project.config.json` 为 `project.private.config.json`，填入你的 AppID

3. **导入开发者工具**
   - 打开微信开发者工具 → 导入项目 → 选择本仓库根目录
   - 后端服务选择"微信云开发"

4. **开通云开发**
   - 顶部菜单「云开发」→ 创建环境
   - 将环境 ID 配置到 `miniprogram/app.js` 的 `env` 字段

5. **编译运行** ✅
