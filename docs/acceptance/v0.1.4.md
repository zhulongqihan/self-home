## 🎯 端到端流程
```
微信小程序启动
  → 自动跳到 pages/launch/index
  → 检查 token（缓存里有就先 verify）
  → 没 token 或失效 → wx.login 拿 code → POST /api/auth/login
  → 后端调微信换 openid → 比对白名单
       ├─ owner   → reLaunch 到 pages/owner/index/index
       ├─ customer → reLaunch 到 pages/customer/index/index
       └─ 陌生人  → reLaunch 到 pages/launch/forbidden（显示 openid）
```

## 🟢 验收点（**这次能看到东西了**）

### 步骤 1：拉取最新代码
本地代码已自动同步，直接打开微信开发者工具即可。

### 步骤 2：编译运行
- 微信开发者工具 → 重新编译（Ctrl/Cmd + B）
- 预期：先看到「🍵 正在打开店铺...」的加载页
- 几秒后：你的 openid 会出现在「店铺暂未营业」兜底页（因为白名单还没填你的 openid）

### 步骤 3：填白名单（关键！）
- 在兜底页**点击 openid 标识，自动复制**
- 把你这个 openid 当 OWNER 填进服务器：
  ```bash
  ssh root@118.31.221.81
  vi /opt/couple-app/.env
  # 改 OWNER_OPENID=你刚才复制的字符串
  pm2 restart couple-app
  ```
- ⚠️ **需要清空数据库已有的空白名单**，重新触发种子注入：
  ```bash
  ssh root@118.31.221.81
  mongosh couple_app --eval 'db.config.updateOne({_id:"global"}, {$set:{"whitelist.owner_openid":"你的openid"}})'
  ```
  （上面这条命令一次到位，不需要重启）

### 步骤 4：重新编译小程序
- 微信开发者工具 → 重新编译
- 预期：进入 **「🧑‍🍳 店长工作台」** 页面 🎉

### 步骤 5：让女朋友扫码上线
- 微信开发者工具 → 真机预览（顶部菜单）
- 让她扫码进入
- 她会出现在「店铺暂未营业」兜底页 → **拍照发她的 openid 给你**
- 你用同样的方法把她的 openid 填到 `whitelist.customer_openid`
- 她重新打开 → 进入 **顾客首页**

### 步骤 6：测试 "退出登录"
- 点 owner 端的「退出登录」→ 确认 → 应该回到 launch 页 → 自动重新登录
