## 🧪 如何导入微信开发者工具

### 前置：下载并安装微信开发者工具
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 导入步骤
1. 打开「微信开发者工具」→ 用微信扫码登录
2. 左侧选择「**小程序**」→ 右上角点击「**+ 导入项目**」
3. 填写表单：
   - **目录**：项目根目录（不是 miniprogram 子目录！）
   - **AppID**：从 `project.private.config.json` 复制
   - **项目名称**：随便填
   - **开发模式**：小程序
   - **后端服务**：微信云开发
4. 点击「确定」

### 首次进入后
1. 顶部菜单「**云开发**」→ 弹窗提示开通 → 选「**按量付费**」（免费额度内不扣费）
2. 创建环境：名称随便填 → 复制环境 ID
3. 打开 `miniprogram/app.js`，把 `env: 'DEFAULT'` 改为你的环境 ID
4. 顶部「**编译**」→ 模拟器应显示「🚧 首页 页面 - 待开发」即为成功
