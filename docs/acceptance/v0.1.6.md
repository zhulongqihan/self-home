# 验收清单 v0.1.6 - 任务 6：商品分类 + 下单闭环

## 本次改动
- 后端新增分类/商品/订单模型与接口
- 启动时自动种子默认分类与商品
- 顾客端实现商品列表、商品详情、购物车、订单列表
- 顾客首页入口接入真实商品页和购物车

## 如何体验
1. 启动后端并确保种子执行成功
2. 微信开发者工具重新编译小程序
3. 顾客登录后从首页进入商品列表
4. 进入任意商品详情，选择规格与数量，加入购物车或立即下单
5. 在购物车提交订单后，进入订单页查看刚创建的订单

## 验收点（请逐项打勾）
- [ ] 商品分类可切换，分类下商品正确展示
- [ ] 商品详情可显示图片、规格、数量选择
- [ ] 购物车数量增减与合计金额正确
- [ ] 提交订单成功并返回订单号
- [ ] 订单列表能看到最新订单及状态
- [ ] 全流程无明显报错（前端控制台 + 后端日志）
