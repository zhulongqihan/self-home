# 验收清单 v0.1.9 - 店长端商品 CRUD

## 本次改动
- 后端：店长商品 `GET/POST/PUT/PATCH/DELETE`（含 `GET /api/products/owner/all`）
- 店长端：商品列表（筛选分类/在售状态）、新建/编辑、上下架、删除
- 工作台「商品管理」入口可用

## 如何体验
1. 开发者工具重新编译
2. 店长 `yangyang` 登录 → 工作台 → **商品管理**
3. 点「＋ 新建商品」：填名称、分类、价格，选图片风格或 URL，保存
4. 顾客 `zhuzhu` → 厨房 Tab：应能看到新商品（在售状态）
5. 回到店长：对商品「下架」，顾客厨房应不再显示
6. 试编辑商品、删除商品
