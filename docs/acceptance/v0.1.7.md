# 验收清单 v0.1.7 - 顾客端 IA + 厨房点单页

## 本次改动
- PRD 升级 v2.3
- 欢迎页（一句话 + 一张图，店长可改）
- 四 Tab：厨房 / 订单 / 发现 / 我的（文案可配，自定义 TabBar）
- 厨房：左分类 + 右商品列表 + 悬浮购物车（实时合计、抽屉下单）
- 商品图：`image_style` 占位（chikawa/线条小狗/布布/一二）或自定义 URL

## 如何体验
1. 微信开发者工具重新编译
2. 顾客登录 `zhuzhu` → 应先看到欢迎页
3. 点击进入 → 默认在「厨房」Tab
4. 左栏切换分类，右侧商品列表更新；点「选规格」或「+」加购
5. 底部悬浮条显示件数与合计；点击展开抽屉，提交订单
6. 切换「订单」「发现」「我的」Tab
7. 店长 `yangyang` → 设置页可改欢迎页文案与图片 URL
